Product Service - Variant Management API

DOC-API-CONTRACTS-PRODUCT-SERVICE-API-POST-VARIANTS | 2026-06-09
api product-service

Variant CRUD and inventory management endpoints. All endpoints require SELLER JWT and product ownership. Base URL: `/api/v1` Service: product-service (Port 8084)

  • Spec version
    5.4.0
  • Server(s)
    https://api.marketplace.vn/api/v1, http://localhost:8084/api/v1
  • Source
    api-contracts/product-service/api-post-variants.yaml

Operations

MethodPathSummary
GET/seller/products/{productId}/variantsList variants of a product
POST/seller/products/{productId}/variantsCreate a new variant (SKU)
PUT/seller/variants/{variantId}Update variant
DELETE/seller/variants/{variantId}Delete variant
GET/inventory/{skuCode}Check stock by SKU
PUT/inventory/{skuCode}/restockAdd stock (seller)
POST/seller/inventory/adjustAdjust stock (delta)
