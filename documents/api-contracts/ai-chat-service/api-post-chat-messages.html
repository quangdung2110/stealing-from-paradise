AI Chat Service API

DOC-API-CONTRACTS-AI-CHAT-SERVICE-API-POST-CHAT-MESSAGES | 2026-06-09
api ai-chat-service

AI-powered chat support with multi-turn conversations, tool calls, and human-in-the-loop confirmation for high-risk actions. AI Chat Service (Port 8093). Spring AI + PostgreSQL + Redis + Kafka.

  • Spec version
    5.5
  • Server(s)
    http://localhost:8093/api/ai
  • Source
    api-contracts/ai-chat-service/api-post-chat-messages.yaml

Operations

MethodPathSummary
POST/sessionsCreate a new chat session
GET/sessionsList user's chat sessions
DELETE/sessions/{sessionId}Close a chat session
POST/chatSend message and receive streaming AI response
GET/chat/historyGet message history for a session
POST/confirmConfirm or reject a pending action
GET/suggestGet quick suggestion questions
